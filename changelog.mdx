## February 26, 2026

## New Features

**Google/Vertex AI Code Execution Tool**
Support for Python code execution in a sandboxed environment through Google and Vertex AI models.
- Execute Python code directly within model conversations
- Automatically capture and display code execution results
- Available alongside existing tools like web search and maps

**Enhanced Markdown Rendering**
Improved markdown support across the platform for better content formatting.
- Richer text formatting in messages and responses
- Better display of structured content in conversations
