The primary way to access PromptLayer is through our [Python](/languages/python) or [JavaScript](/languages/javascript) SDK libraries. The SDKs are built on top of our REST API.

If you use another language, you can interact directly with the REST API.

Use [Resolve Folder ID by Path](/reference/resolve-folder-id) to look up an ID, or [Create Folder](/reference/create-folder) to make one. See [Unified Registry](#unified-registry).
</Tip>
