---
title: "Create Table Sheet File Import"
openapi: "POST /api/public/v2/tables/{table_id}/sheets/{sheet_id}/imports/file"
description: "Import a CSV file into an existing Table sheet."
---

Start an asynchronous CSV import into an existing Table sheet.

Send `file_name` ending in `.csv` and `file_content_base64` containing the base64-encoded CSV content. The response returns an `operation_id` and `status_url` for polling with [Get Table Sheet Import Operation](/reference/get-table-sheet-operation).
